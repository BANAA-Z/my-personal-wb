# CSS 学习笔记

## 一、CSS 基础认知

CSS 全称层叠样式表，主要作用是对 HTML 页面进行样式设置、布局排版，实现页面的美化与视觉统一，将内容与样式分离，让网页结构更清晰，维护更便捷。

## 二、CSS 引入方式

**行内样式**：直接在 HTML 标签内通过 style 属性编写样式，仅作用于当前标签，适合临时简单样式修改，不便于统一管理。

**内部样式表**：在 HTML 页面的 head 标签中，使用 style 标签编写所有样式，仅对当前页面生效，适合单页面样式编写。

**外部样式表**：单独创建.css 文件，在 HTML 中通过 link 标签引入，是最推荐的方式，可实现多页面样式共享，便于批量修改和维护。

## 三、CSS 选择器

**元素选择器**：直接使用 HTML 标签名作为选择器，比如 p、div、h1，会选中页面内所有该标签元素。
类选择器：以。开头，自定义类名，可给多个标签设置同一个类，实现样式复用，灵活性最高。

**ID 选择器**：以 #开头，ID 具有唯一性，一个页面只能有一个相同 ID，适合单独设置某个特定元素的样式。

**通配符选择器**：用 * 表示，选中页面所有元素，常用于清除默认内外边距。

**后代选择器**：通过空格分隔，选中父元素下的所有子元素、孙元素，层级选择精准。

## 四、CSS 核心样式属性

### （一）文本样式

**color**：设置文字颜色，可使用英文单词、十六进制、RGB 值。

**font-size**：设置文字大小，常用单位 px、em、rem。

**font-weight**：设置文字粗细，normal 正常、bold 加粗。

**text-align**：设置文字水平对齐方式，left 左对齐、center 居中、right 右对齐。

**line-height**：设置行高，控制文字行间距，实现文字垂直居中。

**text-decoration**：设置文字装饰，none 无装饰、underline 下划线、line-through 删除线。

### （二）盒子模型

盒子模型是 CSS 布局核心，所有元素都可看作盒子，包含四个部分：内容区 content、内边距 padding、边框 border、外边距 margin。

**padding**：元素内容与边框之间的距离，上下左右可单独设置。

**border**：元素边框，需设置宽度、样式、颜色，比如 border:1px solid #000。

**margin**：元素与元素之间的距离，上下左右可单独设置，水平居中可设置 margin:0 auto。

### （三）布局相关样式

**display 属性**：控制元素显示方式

**block**：块级元素，独占一行，可设置宽高，比如 div、p、h 标签。

**inline**：行内元素，不独占一行，无法设置宽高，比如 span、a 标签。

**inline-block**：行内块元素，兼具两者特点，不独占一行且可设置宽高。

**none**：隐藏元素，不占据页面空间。

**Flex 布局**：现代常用布局方式，通过 display:flex 开启，可轻松实现元素水平垂直居中、均匀分布，适配多场景排版，核心属性有 justify-content（水平对齐）、align-items（垂直对齐）、flex-direction（排列方向）。

## 五、CSS 变量与主题切换

CSS 变量可自定义属性，实现样式统一管理，方便切换主题。在:root 中定义全局变量，格式为 -- 变量名：属性值，使用时通过 var (变量名) 调用。结合 JS 给 body 添加或移除类，切换不同主题下的变量值，实现日间、夜间模式一键切换，还可通过 localStorage 存储主题状态，页面刷新后保持不变。

## 六、过渡与动画

**过渡 transition**：让元素样式变化更平滑，设置过渡属性、时长、速度曲线，比如 hover 时的颜色、位置变化，提升页面交互质感。

**动画 @keyframes**：自定义动画关键帧，设置不同阶段的样式，配合 animation 属性调用，实现循环、淡入淡出等动态效果。

## 七、响应式布局

通过媒体查询 @media，根据不同屏幕宽度设置不同样式，适配电脑、平板、手机等设备，保证页面在不同设备上都能正常显示，避免布局错乱，常用断点 768px 区分移动端与桌面端。

## 八、常见问题与技巧

**清除默认样式**：页面加载时 HTML 标签有默认内外边距，可用通配符 * 设置 margin:0;padding:0 清除。

**元素居中**：水平居中用 text-align:center（行内元素）、margin:0 auto（块级元素）；垂直居中结合 Flex 布局最简便。

**样式优先级**：行内样式＞ID 选择器＞类选择器＞元素选择器，优先级高的样式会覆盖优先级低的。
